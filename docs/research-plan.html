1. Overview

This plan focuses on the generative design agent: building a conditional molecular generator, strong baselines, and an evaluation framework that feeds into the broader agentic discovery system.

Core Scientific Thesis

Single-molecule electronics is a sparse-data, mechanism-rich, experimentally expensive quantum-transport discovery problem. The system should be judged by whether it can:

  1. Propose explicit quantum-interference hypotheses
  2. Design molecules that test those hypotheses
  3. Distinguish new mechanisms from known expert rules
  4. Select positive, negative, and control molecules prospectively
  5. Explain successes and failures
  6. Build an auditable reasoning trail

2. Month 1: Data Foundation & Baselines May 26 — Jun 26

  • Audit SMEMol dataset: deduplicate, validate SMILES, standardize canonical SMILES and SELFIES
  • Verify anchor-group labels (thiol, amino, pyridyl, etc.) for every molecule
  • Assign scaffold IDs and analog-series labels
  • Tag data tier (A/B/C) and split assignment (scaffold split preferred)
  • Export to data/processed/smemol_standardized.csv

3. Month 2: Conditional Graph Generator Jun 26 — Jul 26

Week 5–6: Molecular Graph Representation and Diffusion Setup

  • Choose framework: DiGress (discrete graph diffusion) or MiDi (molecular graph diffusion)
  • Implement SME-specific featurization: S, N, O, Au-binding atoms, anchor-group tokens, scaffold-level graph features
  • Set up training pipeline with scaffold-split train/val/test
  • Train unconditional model; evaluate basic generation quality vs baselines
  • Implement conditional generation: property regression head, classifier-free guidance
M4: Unconditional graph diffusion running, generating valid SME molecules

Week 7–8: Conditioning and Property Control

  • Train conditional model with multi-property conditioning:
    $$\mathbf{c} = [\text{DQI}, S, \Delta G_{\text{reg}}, \eta_{\text{reg}}, \text{SAscore}, \text{scaffold}]$$
  • Measure property controllability:
    $$\text{Controllability}(p) = \mathbb{E}\left[|c_p - \hat{y}_p(G_{\text{gen}})|\right]$$
  • Implement scaffold-conditioned generation (generate analogs of a given scaffold)
  • Anchor-group validity filter: post-hoc check for exactly 2 valid anchor groups
M5: Conditional generator with quantitative property control

Week 9: REINVENT4-Style RL Baseline

  • Implement REINVENT4-style baseline:
    $$\pi_\theta(a \mid s) \leftarrow \text{RL}\left(\text{Transformer}, \; R(x) = w_1 \cdot \text{SMEPre}(x) + w_2 \cdot \text{SAscore}(x) + w_3 \cdot \mathbf{1}[\text{anchor valid}]\right)$$
  • Run head-to-head against conditional graph diffusion under matched compute budget
M6: Sixth baseline operational; first head-to-head comparison

4. Month 3: Agentic Loop & Prospective Validation Jul 26 — Aug 26

Week 10–11: Hypothesis-Conditioned Generation

  • Integrate with hypothesis agent: translate hypotheses to molecular constraints
  • Implement constraint-to-generation mapping:
    $$\text{Hypothesis } \mathcal{H} \xrightarrow{f_{\text{map}}} \{\text{scaffold}, \text{substituents}, \text{anchors}, \text{property targets}\} \xrightarrow{g_{\text{gen}}} \text{Molecules}$$
  • Generate molecules conditioned on 3–5 specific quantum-interference hypotheses
  • Critic agent review: filter generated candidates through the reviewer checklist
M7: End-to-end hypothesis → generation → critique pipeline working

Example Hypotheses to Test

  1. Asymmetric heteroatom placement shifts antiresonance closer to the Fermi level
  2. Cross-conjugated donor–acceptor motifs amplify gate sensitivity
  3. Non-classical anchor alignment creates hidden destructive QI channels
  4. Heteroaromatic fusion breaks expected magic-ratio behavior
  5. Anchor-induced level alignment can dominate over classical connectivity rules

Week 12: Active-Learning Round 1 and Benchmark

Run the full active-learning loop at small scale:

Generate 50,000 molecules
  ↓ Validity / anchor / chemistry filters
  ↓ SMEPre surrogate scoring + uncertainty
  ↓ Diversity and novelty selection
  ↓ Hypothesis and critic-agent review
  ↓ Retrosynthesis / synthetic feasibility filter
  ↓ Select 50 molecules for DFT+NEGF

Selection Policy (per DFT round)

ClassFractionPurpose

Exploration40%Novel scaffolds or high uncertainty

8. System Architecture

Literature Agent → Hypothesis Agent → Design Agent → Quantum-Transport Agent
                                                    ↓
                                              Synthesis Agent
                                                    ↓
                                               Experiment
                                                    ↓
                                   Critic Agent → Revised Hypotheses

The key advance is not merely using a newer generator. The old pipeline:

Generate → Predict → Validate → Update

is standard in AI-for-science by 2026. The agentic framing is what makes this project methodologically novel.
