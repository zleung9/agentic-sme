SME-Agent — 3-Month Generative Research Plan

May 26 — Aug 26, 2026  |  v2  |  Rendered with $\mathrm{\LaTeX}$

Month 1: Data & Baselines May 26 — Jun 26

Weeks 1–2: Dataset

  • Audit SMEMol: deduplicate, validate SMILES, standardize canonical SMILES + SELFIES
  • Verify anchor-group labels; assign scaffold IDs and data tiers (A–F)
  • Export data/processed/smemol_standardized.csv; document gaps
M1: Clean dataset commit

Week 3: Legacy baselines

  • Re-run LSTM generator + SMEPre predictor
  • SELFIES Transformer baseline
  • Random SME-like enumeration (fragment pool + SMILES filter)
M2: 3 baselines running

Week 4: Evaluation + expert rules

  • Unified CLI evaluation harness (10+ metrics)
  • Expert-rule baseline: QI-motif catalog + anchor enumeration
  • Property-conditioning schema defined
M3: 5 baselines benchmarked

Month 2: Graph Generator Jun 26 — Jul 26

Weeks 5–6: Diffusion setup

  • Choose DiGress or MiDi; SME-specific featurization (S, N, O, Au-binding atoms, anchor tokens)
  • Scaffold-split train/val/test; train unconditional model
  • Add classifier-free guidance + property regression head
M4: Graph diffusion generating valid SME molecules

Weeks 7–8: Property control

  • Multi-property conditioning: $\mathbf{c} = [\text{DQI}, S, \Delta G_{\text{reg}}, \eta_{\text{reg}}, \text{SAscore}, \text{scaffold}]$
  • Controllability metric: $\mathbb{E}[|c_p - \hat{y}_p(G_{\text{gen}})|]$
  • Scaffold-conditioned generation; anchor-validity post-filter
M5: Conditional generator with quantitative control

Week 9: REINVENT4 baseline

  • Transformer agent + $R(x) = w_1\text{SMEPre} + w_2\text{SAscore} + w_3\mathbf{1}[\text{anchor}]$
  • Head-to-head vs graph diffusion under matched compute
M6: 6th baseline; first comparison

Month 3: Agentic Loop Jul 26 — Aug 26

Weeks 10–11: Hypothesis-conditioned

  • $\mathcal{H} \xrightarrow{f_{\text{map}}} \{\text{scaffold}, \text{substituents}, \text{anchors}, \text{targets}\} \xrightarrow{g_{\text{gen}}} \text{Molecules}$
  • Generate for 3–5 QI hypotheses; critic-agent review
M7: Hypothesis → generation → critique pipeline

Week 12: Active-learning round 1

  • Generate 50k → filter → SMEPre score → select 50 (40% exploit / 40% explore / 20% controls)
  • Matched-budget benchmark: 6 generators
M8: Benchmark table; DFT batch ready

Week 13: Figure 3

  • DFT hit rate, scaffold novelty, property controllability, hypothesis confirmation
  • Draft Figure 3 + benchmark report; plan round 2
M9: Figure 3 draft
Agentic hypothesis-guided > Non-agentic graph diffusion > REINVENT4-style RL > Expert rules > Legacy LSTM > Random enumeration
